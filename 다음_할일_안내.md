## 🔴 내가 꼭 직접 해야 하는 것

### 1. Python 환경 준비 + 패키지 설치
프로젝트 폴더에서 터미널 열고:
```
pip install -r requirements.txt
```
(Python 3.10+ 필요. 없으면 python.org 에서 설치)

### 2. OpenAI API 키 넣기  ← 가장 중요
- 운영진이 팀 채널로 준 **API 키**를 받으세요.
- `.env.example` 파일을 복사해서 `.env` 로 이름 변경:
  ```
  copy .env.example .env
  ```
- `.env` 열어서 `OPENAI_API_KEY=` 뒤에 실제 키 붙여넣기.
- ⚠️ 이 키는 절대 GitHub·카톡·다른 AI에 붙여넣지 마세요. ($20 한도, 유출 시 즉시 정지)

### 3. 인덱스 구축 실행 (최초 1회)
```
python -m scripts.build_index
```
→ 임베딩 비용이 조금 발생하지만 text-embedding-3-small 이라 매우 저렴(수 센트).

### 4. 질문 테스트
```
python -m scripts.ask
```
→ 잘 나오면 성공. 이후는 튜닝/실험 단계.

### 5. GitHub 레포 만들고 올리기
- 레포 생성 → 초대 없이 개인 레포로.
- `.gitignore` 가 데이터·키를 자동 제외하니 그대로 push 하면 안전.
